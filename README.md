# Getting Started

## Install Dependencies

1. Using Node.js v20.5.1:

```shell
npm i
```

2. Install .NET 6 - [Link](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)

3. Run

```shell
npm start
```
